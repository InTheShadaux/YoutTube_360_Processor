# YoutTube_360_Processor
Video Processor for 360 videos.

1. Download the code in a zip file
2. Extract the contents to a folder.
3. Extract the __main__.zip file.
4. Open the folder where you extracted __main__.zip to.
5. Run __main__.exe
6. Select the output folder.
7. Click Process Video and select the video.
8. Selecting the video starts the process.
9. Check the output folder once processing is complete.
